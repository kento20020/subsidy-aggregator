何をしているサイトか

日本国内の補助金・助成金情報（現在は jGrants 公開API由来）を毎日自動で収集し、 見やすくまとめた静的サイトです。

更新頻度

毎朝 8:00 JST に GitHub Actions が自動でデータを取得・再ビルドします。 最終更新: 2026-05-13 11:56 UTC（50 件掲載）

仕組み

  • jGrants 公開API から公募中の補助金一覧を取得
  • 前回データと SHA256 ハッシュで差分検出
  • 新着・更新分のみ AI で業種・用途・要約をタグ付け
  • Jinja2 で静的HTMLを生成し GitHub Pages に公開

免責事項

  • 掲載情報は参考情報であり、申請可否・採択を保証するものではありません。
  • 最新の正確な情報は必ず各公式サイトでご確認ください。
  • 情報の誤り・欠落・遅延等による損害について責任を負いません。

個人情報の取扱い

本サイトはユーザー登録・ログイン機能を持たず、個人情報を取得しません。 アクセスログ等は GitHub Pages の標準仕様に準じます。
