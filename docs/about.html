このサイトについて

日本国内の補助金・助成金（jGrants 公開API由来）を毎朝自動で収集し、 読まなくても3秒で判断できる形 に整理した静的サイトです。 現在 50 件 掲載中。

このサイトの特徴

  • 広告なし・会員登録不要 — トラッキング目的のCookieも使いません
  • AIが要点を整理 — 公式PDFを読まなくても「対象者・対象経費・必要書類・注意点」が一目で分かります
  • 毎朝自動更新 — GitHub Actions が 8:00 JST に更新を検知
  • 既読カードは自動でグレーアウト — 新着・更新分だけが目立つ仕組み
  • オープンソース — コード・データ取得元すべて公開（GitHub リポジトリ）

使い方

1. 質問形式で絞り込む（おすすめ・初回向け）

初回アクセス時に4つの質問が表示されます。業種・規模・用途・地域を選ぶだけで、該当する補助金だけがリスト化されます。自由入力欄にビジネスの内容を書いて AI に解釈させることもできます。

あとから条件を変えたい場合: 絞り込み欄の「条件を質問形式で設定する」リンクから再表示できます。

2. フィルタで直接絞り込む

業種・規模・用途・地域・締切で絞り込めます。「並び順」で締切順・補助額順に並び替えも可能。条件は自動でブラウザに保存され、再訪問時に復元されます。

3. URL を共有する

絞り込み条件は URL に反映されます。「この条件のURLをコピー」ボタンで、自分の条件をそのまま他の人に渡せます。

4. 詳細ページで判断する

カードの「詳細を見る」から個別ページへ。最上部の 「3秒判断材料」 セクションに、対象者・対象経費・必要書類・注意点が抽出されています。「申請ページを公式サイトで開く」ボタンから直接 jGrants に遷移できます。

5. 既読をグレーアウト・未読のみ表示

一度開いた補助金カードは自動でグレー表示されます。「未読のみ表示」をオンにすると、まだ見ていない補助金だけが残ります。リセットも可能。

更新頻度

毎朝 8:00 JST に GitHub Actions が自動でデータを取得・再ビルドします。最終更新: 2026-05-14 00:01 UTC

仕組み

  1. jGrants 公開API から公募中の補助金一覧を取得
  2. 前回データと SHA256 ハッシュで差分検出（新着 / 更新 / 据置 を判定）
  3. 新着・更新分のみ詳細APIで本文を取得
  4. Claude Haiku が本文から「対象・経費・書類・注意点」を構造化抽出
  5. Jinja2 で静的HTMLを生成し GitHub Pages に公開

免責事項

  • 掲載情報は参考情報であり、申請可否・採択を保証するものではありません。
  • AI による要点抽出は誤りを含む可能性があります。最終的な要件・申請可否は必ず公式サイトでご確認ください。
  • 情報の誤り・欠落・遅延等による損害について責任を負いません。

個人情報の取扱い

本サイトはユーザー登録・ログイン機能を持たず、個人情報を取得しません。 詳細は プライバシーポリシー をご覧ください。
