このサイトについて

プライバシーポリシー

本サイト (補助金まとめ) は、ユーザーの個人情報を取得・保存しません。本ページは、収集していない情報の範囲と、ブラウザ機能・第三者サービスの利用について明示するものです。

1. 個人情報の取得について

本サイトはユーザー登録機能・ログイン機能・お問い合わせフォームを持たず、以下の情報を取得しません。

  • 氏名・メールアドレス・電話番号などの直接識別情報
  • 事業者名・住所などの間接識別情報
  • クッキー (Cookie) を利用した広告配信・行動追跡

2. ブラウザに保存される情報 (localStorage)

サービス利便性のため、以下の情報を ご自身のブラウザ内のみ に保存します。サーバーには送信されません。

  • フィルタ条件: 業種・規模・地域などの絞り込み選択 (再訪問時の自動復元用)
  • ウィザード完了フラグ: 初回ウィザードを表示するかの判定用
  • 既読カードID: 「未読のみ表示」機能のため (補助金IDのリスト)

これらはブラウザの設定で「サイトデータの削除」を実行することでいつでも消去できます。本サイト内の「リセット」ボタンからも削除可能です。

3. 第三者サービスの利用

3.1 GitHub Pages

本サイトは GitHub Pages 上でホストされており、GitHub 社のサーバーログ (IPアドレス、User-Agent、参照元など) が標準仕様により記録される場合があります。詳細は GitHub プライバシーステートメント をご参照ください。

3.2 Cloudflare Workers (任意機能)

ウィザードの自由入力欄でテキストを送信した場合のみ、入力テキストが Cloudflare Workers 経由で Anthropic の Claude API に送られ、業種・規模等のフィルタ値に変換されます。テキストは Worker 上で一時的に処理されるのみで、保存・記録されません。Cloudflare のプライバシー方針は こちら。

3.3 jGrants 公開API

掲載している補助金情報はデジタル庁が提供する jGrants 公開API から自動取得しています。ユーザーが本サイトを閲覧することによって jGrants 側にユーザー情報が送られることはありません。

4. アクセス解析

現時点では Google Analytics 等のアクセス解析ツールを導入していません。今後導入する場合は、クッキー不要・匿名集計のサービス (例: Cloudflare Web Analytics) のみを採用し、本ページに追記します。

5. 免責事項

  • 掲載情報は jGrants 公開API由来の公開情報を機械的に整理したものです。
  • AI による要点抽出は誤りを含む可能性があります。最終的な要件・採択可否は必ず公式サイトでご確認ください。
  • 本サイトの利用によって生じた直接的・間接的損害について責任を負いません。

6. 改訂・お問い合わせ

本ポリシーは予告なく改訂されることがあります。ご質問・ご指摘がある場合は GitHub Issues よりお知らせください。
